# Credit-scoring-project
Credit Risk Scoring
Описание проекта

Проект посвящён задаче оценки кредитного риска — предсказанию вероятности дефолта потенциального заёмщика (credit scoring).

Решение данной задачи представляет практическую ценность для кредитных организаций при принятии решений о выдаче кредита, а также может быть полезно заёмщикам для оценки собственной платёжеспособности.

Данные

В качестве данных используется датасет Give Me Some Credit с платформы Kaggle:
https://www.kaggle.com/c/GiveMeSomeCredit

Подход и методы

В качестве метрики качества была выбрана ROC-AUC, так как она устойчива к дисбалансу классов и широко применяется в задачах кредитного скоринга.

В ходе работы были выполнены:

предобработка данных (работа с пропусками и выбросами);

анализ и визуализация данных;

исследование взаимосвязей между признаками;

работа с дисбалансом классов;

обучение и оптимизация моделей машинного обучения.

Были рассмотрены следующие модели:

Logistic Regression

Random Forest

Gradient Boosting (XGBoost)

По итогам сравнения и интерпретации результатов лучшей моделью оказался градиентный бустинг.

Особое внимание уделялось интерпретации результатов модели и анализу значимости признаков.

Результаты

Лучшая модель достигла значения ROC-AUC = 0.87, что свидетельствует о высокой способности модели различать надёжных и ненадёжных заёмщиков.

Наиболее значимыми оказались признаки, связанные с предыдущими невыполнениями кредитных обязательств.

Дальнейшие улучшения

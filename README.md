# Credit-scoring-project
Проект посвящён задаче оценки кредитного риска — предсказанию вероятности дефолта потенциального заёмщика (credit scoring). 
Решение данной задачи представляет практическую ценность для кредитных организаций при принятии решений о выдаче кредита, а также может быть полезно заёмщикам для оценки собственной платёжеспособности. 

В качестве данных используется датасет Give Me Some Credit с платформы Kaggle: https://www.kaggle.com/c/GiveMeSomeCredit 

В качестве метрики качества была выбрана ROC-AUC, так как она устойчива к дисбалансу классов и часто применяется в задачах кредитного скоринга. 
В ходе работы были обработаны, проанализированы и визуализированы данные, а также выявлены некоторые закономерности в данных и между признаками. Затем были обучены и оптимизированы различные модели (LogisticRegression, RandomForest, XGBoost). По итогам сравнения и интерпретации результатов лучшей моделью оказался градиентный бустинг. 
Особое внимание уделялось работе с дисбалансом классов и интерпретации результатов модели. 

В результате была обучена модель, достигшая значения ROC-AUC = 0.87, что означает высокую способность модели различать надёжных и ненадёжных заёмщиков. Самыми значимыми признаками оказались признаки, связанные с предыдущими невыполнениями кредитных обязательств. 

В заключении проекта рассмотрены возможные направления для дальнейшего улучшения модели и её качества.

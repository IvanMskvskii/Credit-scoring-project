# Credit-scoring-project
Проект посвящён задаче оценки кредитного риска — предсказанию вероятности дефолта потенциального заёмщика.
Решение данной задачи представляет практическую ценность для кредитных организаций при принятии решений о выдаче кредита, а также может быть полезно заёмщикам для оценки собственной платёжеспособности.

В качестве данных используется датасет Give Me Some Credit с платформы Kaggle:
https://www.kaggle.com/c/GiveMeSomeCredit

В результате была обучена модель, достигшая значения ROC-AUC = 0.87, что означает высокую способность модели различать надёжных и ненадёжных заёмщиков.

В заключении проекта рассмотрены возможные направления для дальнейшего улучшения модели и её качества.
